# AI PDF Summarization App

This project is an AI-powered application that generates summaries from PDF documents using Large Language Models.

The application is built using LangChain, OpenAI models, and Streamlit for the user interface.

## Features

- Upload or load a PDF file
- Generate AI-powered summaries
- Custom prompt-based summarization
- Multiple summarization chain methods
- Adjustable chunk size and overlap
- Supports ChatGPT and GPT-4 models

## Technologies Used

- Python
- Streamlit
- LangChain
- OpenAI API
- PyPDF

## Installation

Clone the repository

git clone https://github.com/JYOTHIkudipudi/Summarization-App

Move into the project directory

cd pdf-summarizer-langchain-streamlit

Install dependencies

pip install -r requirements.txt

## Setup

Set your OpenAI API Key

export OPENAI_API_KEY="your_api_key"

## Run the Application

streamlit run app.py

## Use Case

This tool helps students, researchers, and professionals quickly summarize long PDF documents using AI.
